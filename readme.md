# **ZenSMS - Privacy-focused SMS app for Android**

ZenSMS is built for power users who value privacy and control. No cloud. No tracking. No BS.

📲 **[Download on Google Play Store](https://play.google.com/store/apps/details?id=com.zensms.app)**

---

## 🐛 Report a Bug

Found a bug? Please [open an issue](https://github.com/Essential-Labs8/ZenSMS-public/issues/new) with:
- What happened (expected vs actual behavior)
- Steps to reproduce
- Android version and device model
- Screenshots if relevant

## ✨ Request a Feature

Have an idea? [Create a feature request](https://github.com/Essential-Labs8/ZenSMS-public/issues/new) and describe:
- What you want
- Why it would be useful
- How you envision it working
