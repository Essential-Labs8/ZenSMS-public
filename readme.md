## 🐛 Report a Bug

Found a bug?
1.  **Search** existing issues to ensure it hasn't already been reported.
2.  [Open an issue](https://github.com/Essential-Labs8/ZenSMS-public/issues/new) with:
    - What happened (expected vs actual behavior)
    - Steps to reproduce
    - Android version and device model
    - Screenshots if relevant

## ✨ Request a Feature

Have an idea?
1.  **Check [Issue #1](https://github.com/Essential-Labs8/ZenSMS-public/issues/1)** to ensure your idea is not on the "Non-Goals" list.
2.  [Create a feature request](https://github.com/Essential-Labs8/ZenSMS-public/issues/new) and describe:
    - What you want
    - Why it improves privacy or utility
    - How you envision it working
