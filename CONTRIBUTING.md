# Contributing to this Project

First off, thank you for being here! We use this repository to track bugs, ideas, and tasks.

## 🚫 No Pull Requests
Please note that this is an **Issues-Only** repository. 
* We do not accept Pull Requests here.
* Any Pull Requests opened will be automatically closed by our bot.
* This setup helps us manage our internal development workflow more efficiently.

## ✅ How to Contribute
Even though we don't accept code here, your input is incredibly valuable! 

### Reporting Bugs
* Search existing issues to see if the bug has already been reported.
* If not, open a new issue using the **Bug Report** template.
* Provide as much detail as possible (screenshots, steps to reproduce, etc.).

### Suggesting Features
* Open a new issue using the **Feature Request** template.
* Describe the "Why" behind the feature—what problem does it solve for you?
